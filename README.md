# Event Stream DSA

A collection of high-performance data structure and algorithm (DSA) implementations
designed to analyze a 250MB e-commerce event stream log—all processed in-memory without a database.

---

## Dataset Information

Due to GitHub's file size limitations (100MB), the primary dataset is not included in this repository.

* **Source:** [Retail Rocket - Cleaned Events Dataset (Kaggle)](https://www.kaggle.com/datasets/cianaellington/retail-rocket-cleaned-events-dataset)
* **File Size:** ~246.53 MB
* **Format:** CSV

### Setup Instructions
1. Download the dataset from the link above.
2. Create a folder named `dataset/` in the root directory.
3. Place the downloaded `clean_events.csv` inside that folder and rename it to `dataset.csv`.
4. Ensure your `.gitignore` includes `dataset/` to avoid push errors.
